Steps to create Continuous Delivery Pipeline

1.GIT REPOSITORY

Created a repo and made an app.js file containing Website Content and pushed the code to remote repo

2.DEPLOYED WEB APP

Configured Elastic Beanstalk with appropriate Environment,Webserver, Platform and HTTP protocol

3.BUILD PROJECT

It compiled source code, run tests, and packaged software for deployment.

4.CREATED DELIVERY PIPELINE

Configured source, build,deploy stages using GitHub repo ,AWS CodeBuild, AWS ElasticBeanstalk application respectively and then deployed application through a pipeline.

5.FINALIZED PIPELINE AND TEST

Added a review stage to this pipeline to Manually approve a change before it is deployed
